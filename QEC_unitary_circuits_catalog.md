# Catalog of Purely-Unitary QEC Circuits for Benchmarking *Manjushri*

## 1. Purpose and framing

This catalog assembles **45 verified quantum-error-correction (QEC) circuits** from a literature sweep, curated as benchmark inputs for **Manjushri**, an equivalence-checking tool for **purely unitary** circuits. Manjushri's scope is **Clifford+T circuits in OpenQASM 2.0** — no measurement, reset, or classical control — and it checks **equivalence up to global phase**. The catalog therefore admits only the *unitary* slices of QEC machinery: **encoders**, **logical-state-preparation circuits**, **transversal and fold-transversal logical gates**, the **Clifford encoder/correction parts of magic-state distillation**, and **stabilizer / graph / cat-state preparation**. Anything that genuinely requires measurement (syndrome extraction, flag verification, gauge-fixing by measurement, post-selection in distillation) is *out of scope* and is included only where a clean unitary sub-circuit can be cleanly extracted (flagged `partial`).

These make excellent equivalence-checking benchmarks because so many of them come with a **second, equivalent form**: a textbook encoder versus an optimized/RL-discovered encoder for the same logical state; a recursive versus a row-reduced construction for the same code; or a **transversality identity** ("apply physical gate `G` to every qubit" = "apply logical gate `G_L`"), which is exactly a two-circuit equivalence (up to global phase) that a checker should validate. The **seed/gold example** is the Chamberland–Cross non-fault-tolerant logical-|H⟩ preparation circuit for the [[7,1,3]] Steane code, **arXiv:1811.00566, Fig. 4** (|H̄⟩_nf = T̄|0̄⟩). It is Clifford+T, 7 qubits, measurement-free — and it has multiple equivalent forms (a re-derived |0_L⟩ encoder + transversal-T, or a library-synthesized stabilizer encoder + transversal T), making it the prototype the rest of this catalog is built around.

> **Note on the seed circuit's availability:** a local OpenQASM 2.0 file **does exist** — `steane_H_nf_fig4.qasm` (7 qubits, H/S/T/CNOT) in the working directory. It was **independently verified by direct statevector simulation** (not merely trusted): the output is a normalized +1 eigenstate of all six Steane stabilizers with ⟨X_L⟩ = ⟨Z_L⟩ = 1/√2, ⟨Y_L⟩ = 0, and ⟨H^⊗7⟩ = +1 — i.e. exactly the logical |H⟩ = +1 eigenstate of logical H, matching Chamberland–Cross Fig. 4 (|H̄⟩_nf = T̄|0̄⟩). It is ready to use directly as a Manjushri input.

All 45 entries were marked **high confidence** by the verifier. The corrections below (fabricated/mis-attributed authors, wrong gate sets, reversed logical actions, wrong figure numbers) have been **applied** — the catalog presents the corrected facts, not the original finder claims.

## 5. Equivalence-pair opportunities (highest-value Manjushri test cases)

Each pair should be **equal up to global phase**.

1. **Recursive QRM(r,m) encoder ≡ row-reduced QRM(r,m) encoder** (entries 8 ≡ 7) — same code, different CNOT counts (e.g. (2,4): 53 vs 30). *Both from `Praveen91299/QRM`.*
2. **Standard Steane encoder ≡ RL-discovered Steane encoder** (0/43 ≡ 5) — same |0_L⟩ stabilizer state.
3. **|0_L⟩ encoder + transversal T̄ ≡ Chamberland–Cross Fig. 4 |H⟩_nf** (0 + T̄ ≡ 12) — the seed-circuit self-consistency check.
4. **encoder · T†^⊗15 ≡ logical-T · encoder** for [[15,1,3]] (entry 10) — the transversality identity (corrected direction: physical T†^⊗15 = logical **T**).
5. **encoder · H^⊗7 · encoder⁻¹ ≡ logical H** for Steane (entry 27); **encoder · H^⊗4 · encoder⁻¹ ≡ logical (H⊗H)·SWAP** for [[4,2,2]] (entry 36).
6. **encoder · (T/T†)^⊗8 ≡ encoder · logical-CCZ** for [[8,3,2]] (entry 34); cross-check the decoded action against the 3-qubit reference (entry 35).
7. **Two distinct code automorphisms realizing the same logical Clifford ≡ each other**, and **physical automorphism circuit ≡ its abstract logical gate** (entry 23, autqec).
8. **Naive/SMT-optimal CSS encoder ≡ recursive encoder** for [[15,1,3]] (entry 39) — MQT QECC vs QRM repo.
9. **Surface-code |0⟩_L: 21-step ≡ 13-step** (entry 26); **local ≡ non-local expansion encoder** (entry 20).
10. **Golay encoder ladder**: 57-CNOT (1106.2190) ≡ 56-gate A* (2503.14660) ≡ 61-gate RL (2402.17761) (entry 22).
11. **BCH encoder ≡ its cyclic/Frobenius-automorphism image** (entry 17) — same circuit under a qubit permutation.
12. **Shor encoder ≡ bit-flip-block-reordered Shor encoder** (entry 2) — equivalence under commuting-block reordering.
13. **|Y⟩ = S|+⟩ ≡ alternate Clifford prep circuits** (entry 44) — minimal global-phase regression fixture.

## 6. Gaps & next steps

- **Genuinely thin: graph-state / cat-state preparation.** Despite being named in scope, the sweep returned **no dedicated GHZ/cat-state or graph-state-prep entry** — only stabilizer-state encoders that incidentally prepare them. *Next:* add canonical n-qubit cat-state ladders (used in Shor-style ancilla/flag prep) and graph-state prep (Hadamard-layer + CZ-by-edges) — both trivially generatable and ideal small fixtures with many equivalent orderings.
- **Distillation is mostly `partial`.** All four distillation entries (22, 37, 41, 42) require excluding a projector/measurement. The *clean* unitary objects are the CSS encoder + transversal-T layer + diagonal Clifford correction. *Next:* concretely build (via MQT QECC + a hand-coded Eq.-10 correction) at least the [[15,1,3]] Bravyi–Haah unitary, and verify the `U·T^⊗n|G⟩` identity.
- **Cross-code / code-conversion is thin and citation-fragile** (entry 15): the unitary circuit lives in arXiv:1511.02596, not the more famous 1403.2734 (which is measurement-based). *Next:* implement the Clifford IABC reduction and add more code-switching unitaries.
- **Larger non-CSS and k>1 encoders are underrepresented** (only [[8,3,3]] at k=3, [[4,2,2]]/[[8,3,2]] at k=2/3). *Next:* generate standard-form encoders for more multi-logical codes to stress the checker beyond k=1 and CSS.
